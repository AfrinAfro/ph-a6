1) What is the difference between var, let, and const?
Answer: var → old way to declare variable, can be changed and redeclared.
let → modern way, can be changed but cannot be redeclared.
const → value cannot be changed after declaration.

2) What is the difference between map(), forEach(), and filter()? 
Answer: map() → creates a new array by changing every element.
forEach() → only loops through array, does not return new array.
filter() → creates a new array with selected elements.

5) Explain template literals in ES6. How are they different from string concatenation?
Answer:Template literals use backticks ` ` and allow variables inside strings easily.

1.Template literals are easier and cleaner.
